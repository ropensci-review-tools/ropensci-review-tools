# List all checks currently implemented

## Description

List all checks currently implemented

## Usage

```r
list_pkgchecks(quiet = FALSE)
```

## Arguments

* `quiet`: If `TRUE`, print all checks to screen. Function invisibly
returns list of checks regardless.
