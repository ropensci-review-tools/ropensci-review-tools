# `get_gh_token`

Get GitHub token


## Description

Get GitHub token


## Usage

```r
get_gh_token(token_name = "")
```


## Arguments

Argument      |Description
------------- |----------------
`token_name`     |     Optional name of token to use


## Value

The value of the GitHub access token extracted from environment
 variables.
