# `srr_stats_roclet`

srr_stats_roclet


## Description

Get values of all `srrstats` tags in function documentation


## Usage

```r
srr_stats_roclet()
```


## Details

Note that this function should never need to be called directly. It only
 exists to enable "@srrstats" tags to be parsed from roxygen2 
 documentation.
