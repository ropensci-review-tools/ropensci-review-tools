# `dl_gh_repo`

Download a GitHub repo to local cache


## Description

Download a GitHub repo to local cache


## Usage

```r
dl_gh_repo(u)
```


## Arguments

Argument      |Description
------------- |----------------
`u`     |     URL of GitHub repository


## Value

Path to locally cached '.zip' version of repository
