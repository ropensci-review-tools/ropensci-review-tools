# Download a GitHub repo to local cache

## Description

Download a GitHub repo to local cache

## Usage

```r
dl_gh_repo(u, branch = NULL)
```

## Arguments

* `u`: URL of GitHub repository
* `branch`: Checkout specified (non-default) branch of repo.
