# `get_branch_from_url`

Get branch from a GitHub URL if non-default branch specified there


## Description

Get branch from a GitHub URL if non-default branch specified there


## Usage

```r
get_branch_from_url(repourl)
```


## Arguments

Argument      |Description
------------- |----------------
`repourl`     |     Potentially with "/tree/branch_name" appended


## Value

Branch as single string.
