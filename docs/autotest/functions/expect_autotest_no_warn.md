# `expect_autotest_no_warn`

expect_autotest_no_warn


## Description

Expect `autotest_package()` to be clear of warnings


## Usage

```r
expect_autotest_no_warn(object)
```


## Arguments

Argument      |Description
------------- |----------------
`object`     |     An `autotest` object to be tested


## Value

(invisibly) The same object
