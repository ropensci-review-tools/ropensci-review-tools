# `expect_autotest_no_err`

expect_autotest_no_err


## Description

Expect `autotest_package()` to be clear of errors


## Usage

```r
expect_autotest_no_err(object)
```


## Arguments

Argument      |Description
------------- |----------------
`object`     |     An `autotest` object to be tested


## Value

(invisibly) The same object
